* 论证推理本质上仅有7种最基本套路
* 结论推出本质上只需要5种排除方法

* 看考点精讲的书 / 视频，理解套路

#### 备考误区

* 用日常思维去阅读理解而不是用逻辑思维去分析题目结构
* 过度追求 “技巧” 而忽略了对题目逻辑结构的分析
* 不总结，闷头刷题，追求做题数量的成就感
* 盲目迷信模拟题、押题

#### 三要三不要

* 不要盲目做题，先看一遍思路，有了思路的概念再做题体会思路
* 不要做模拟题，优先做真题 (从最近的开始做)
* 不要孤立的做题，要思考不同题干文字下隐藏的逻辑结构 (逻辑连词)

### 考官出题套路范例

#### 前真后假思路真题范例    

```
小张说：[如果]我中了500万，我[就]送你套房子，哪个选项为真，能证明小张说的话是[错误]的。   
```
答案：中了500万，没送你房子    

解题思路：     

中了500万 -> 送你套房子     

题干逻辑结构：A -> B         
问使其错误的选项：A and 非B          

```
[只有]具有一定文学造诣[且]具有生物学专业背景的人，[才]能读懂这篇文章。如果上述命题为真，哪个选项不可能为真。    
```

答案：从未学过生物学知识的小李读懂了这篇文章     

解题思路：

文学造诣 and 生物学背景 <- 读懂这篇文章

题干逻辑结构：B and C <- A             
问不可能为真的选项：A and (非B or 非C)              

```
[如果]不注重成本的控制，[那么]就无法提供有吸引力的发射价格。
[如果]想在商业发射市场占据一席之地，[那么]就必须提供有吸引力的发射价格。
若上述论述为真，以下哪个选择一定为假？
```

答案：一个小型火箭即占据发射市场的一席之地，同时又不注重成本的控制      

解题思路：    

1) 一席之地 -> 有吸引力的发射价格   A -> B    
2) 非注重成本 -> 非有吸引力的发射价格 等价于 有吸引力的发射价格 -> 注重成本    
3) 非C -> 非B | B -> C （逆否）    
4) A -> B -> C     
5) 一席之地 -> 有吸引力的发射价格 -> 注重成本    
6) 问不可能为真的选项：A and 非C     

```
[所有]值得拥有专利的产品或设计方案[都是]创新。
[所有]的模仿都[不是]创新。
若上述论述为真，以下哪个选择一定为假？
```

答案：有些值得拥有专利的产品是模仿的。     

解题思路：

1) 拥有专利的产品 -> 创新   A -> B     
2) 模仿 -> 非创新 等价于 创新 -> 非模仿           
3) C -> 非B | B -> 非C （逆否）     
4) A -> B -> C        
5) 拥有专利的产品 -> 创新 -> 非模仿       
6) 问不可能为真的选项：A and 非C

## 形式逻辑基础知识

### 形式逻辑基础

#### 运算符

* 非 (NOT)：非A，非B     
* 或 (OR)：A or B，A or 非B，B or A    
* 且 (AND)：A and B，A and 非B，非B and A    
* 推出 (->)：A -> B，A -> 非B，B <- A (B 被 A 推出)      

1) 选言命题：A 和 非A 是矛盾关系，一定一真一假。若 A 为真，那么 非A 为假，反之亦然。给出一个命题为真，等同于给出它的否命题为假。       
2) 选言命题：A or B 的文字描述：A 或者 B，A 和 B [至少有一个]。A 和 B 至多有一个去 = 至少有一个不去 = 非A or 非B。       
3) 联言命题：A and B 的文字描述："也" -- 我是...， xxx也是；"逗号" -- 我是，xxx是；"并且" -- 收入增高并且幸福感增加；"但是" -- 他胜利了，但他不开心。     

### 选言联言命题真假判断

```
若中国应该大力发展农业为真，
试着判断以下4个命题是真？是假？或者不能确定真假？

1) 中国应该大力发展农业或者大力发展工业    A or B 真    
2) 中国应该大力发展农业并且大力发展工业    A and B 不能确定    
3) 中国不应该大力发展农业，也不应该大力发展工业  非A and 非B 假    
3) 或者中国不应该大力发展农业，或者不应该大力发展工业  非A or 非B 不能确定    
```

### 假言命题真假判断     

#### 定义

##### 充分条件假言命题

即有保证。在逻辑学上：如果 A 的发生一定能保证 B 的发生，那么 A 就叫做 B 的充分条件，逻辑表达式为 A -> B。   

例句：如果摩擦一定会生热    
例句：如果断电电灯泡一定会熄灭    

在语言中一般用下面的逻辑连词来描述：   

[如果] A。。。[那么] B。。。     
[只要] A。。。[就] B。。。    
[若] A。。。[则] B。。。    
[所有的] A，[都是] B。。。  

##### 必要条件假言命题

即必须要有的。    

在逻辑学上，如果要得出 B 条件，A 条件必须成立，但 A 条件成立，也不足以证明 B。     

那么，A 就是 B 的必要条件，表达为 A <- B。    

例句：[要想] 下雨 [必须] 天上有云。  天上有云 <- 下雨 (* 箭头指向必须 *)    
例句：[只有] 有足够氧气，人 [才] 能生存。    足够氧气 <- 人才能生存    
例句：一个国家 [要想] 保持政治稳定，它所创造的财富 [必须] 得到公正的分配。 公正分配 <- 政治稳定
例句：维护好经济秩序，是经济快速发展的 [基础]。   经济秩序 <- 快速发展   

在语言中一般用下面的逻辑连词来描述：        

[只有] A。。。[才] B。。。(*)       
A 对于 B 是 [必须] 的 / [不可缺少] 的    
A 是 B 的 [基础] / [必要条件] / [重要前提]     

##### 充要条件假言命题

在逻辑学上，当 A 同时即是 B 的充分条件，又是 B 的必要条件的时候，A 叫做 B 的充分必要条件，简称为充要条件。     

在语言中一般用下面的逻辑连词来描述：    

A 发生 [当且仅当] B 发生。    

等同于：A -> B 和 B -> A 同时成立。如果 A，那么 B。只有 A，才 B。        

### 假言命题逻辑本质

A -> B 什么情况下为假？   [仅在] A and 非B(前真 and 后假)    前 -- 箭头发起方向    
A <- B 什么情况下为假？   [仅在] 非A and B(前真 and 后假)    前 -- 箭头发起方向     

充分条件描述 A -> B 真值表：    

| A | B   | A -> B   |
|--|-----|-----|
| 真 | 真 | 真 |
| 真 | 假 | 假 |
| 假 | 真 | 真 |
| 假 | 假 | 真 |

```
小张说：如果我中了500万，我就送你套房子。      
小李说：你说的是假话，在骗人。       
哪个选项最能表述小李的意思：    
```

答案：我中了500万，没送你房子。      

必要条件描述 A <- B 真值表：

| A | B   | A <- B   |
|--|-----|-----|
| 真 | 真 | 真 |
| 真 | 假 | 真 |
| 假 | 真 | 假 |
| 假 | 假 | 真 |

```
小张说：这次我们录取的原则是，只有评分优秀，才能够录取。         
小李说：那么针对小王，你没有遵循这个原则。       
哪个选项最能表述小李的意思：    
```

答案：小王评分不优秀，结果被录取了。    

总结：    

不用区分充分还是必要，它只是一个概念，只是主语的角度不同，要区分逻辑的前件与后件。     

前件 -> 后件    
后件 <- 前件     

一个逻辑，仅在同时满足前件为真 (逻辑箭头发起的方向) 和后件为假 (逻辑箭头结束的方向) 的时候，逻辑才为假。       

### 常见逻辑连词

#### 假言命题常见逻辑连词

当需要判断一个逻辑的真假时，关键是判断逻辑箭头的方向，并不需要判断充分 / 必要条件。    

需要做的事：根据逻辑连词，把文字替换成逻辑表达式。    

常见的逻辑连词与其相对应的逻辑箭头方向。       

1) 充分条件表述形式：   

如果 A，那么 B。   描述为： A -> B      
如果 A，则 B。   描述为： A -> B     
所有的 A，都是 B。   描述为： A -> B           
只要 B，就 A。   描述为： B -> A     

2) 必要条件表述形式：   

只有 B，才 A。   描述为：B <- A      
只有 A，才 B。   描述为：A <- B   

3) 特殊逻辑连词：[除] 非 A，否则 B。 

描述为：非 A -> B    

4) 特殊逻辑连词：(否则 B)，除非 A。     

[例子] (否则)肖群一周工作五天，除非这周内有法定节假日。 没有法定节假日 -> 工作五天   

除非不 A，否则 B。 描述为：A -> B    
除非不 A，否则不 B。 描述为：A -> 非 B    
除非 A，否则不 B。 描述为：非 A -> 非 B
B，除非 A。 描述为：非 A -> B    
除非 A，B。 描述为：非 A -> B   

> 优先级：才 > 如果(若) > 除非

### 形式逻辑解题必备能力

只看逻辑连词，不看文字。    

## 形式逻辑四大考点

### 考点一 对逻辑关系加 "非"

1) 非 (非A) = A     
2) 非 (A or B) = 非 A and 非 B    
3) 非 (A and B) = 非 A or 非 B    
4) 非 (A -> B) = A and 非 B [前真后假]   

```
小红想买一件漂亮的衣服，她父亲同意或者她母亲同意都会给她买，可是小红没能说服家人给她买这件衣服，说明：    
```

答案：她父母都不同意给她买    

解题思路：    

非 (父亲同意 or 母亲同意)    

题干逻辑结构：非 (A or B) = 非 A and 非 B          

```
一个项目要实施前必须得到总经理和董事长的签字，这个项目没有得到实施，说明：     
```

答案：或者总经理没签字，或者董事长没签字       

解题思路：    

非 (总经理签字 and 董事长签字)      

题干逻辑结构：非 (A and B) = 非 A or 非 B             

### 考点二 or 和 -> 的相互转化   

对于 or 的逻辑，不管左右是什么，对 or 的左边加非，可以推出右边。对 or 的右边加非，可以推出左边。    

1) A or B = 非 A -> B = 非 B -> A (你掏钱 or 我掏钱 = 非你掏钱 -> 我掏钱 = 非我掏钱 -> 你掏钱)    
2) 非 A or 非 B = A -> 非 B = B -> 非 A (小张和小李至多入选一个 = 小张不如选 or 小李不如选)       
3) 非 A or B = A -> B = 非 B -> 非 A   
4) A or 非 B = 非 A -> 非 B = B -> A     
5) 非 (非 A or B) = 非 (A -> B) = A and 非 B   
6) A and B or C = 非 A or 非 B -> C = 非 C -> A and B    

```
小陈并非既懂英语又懂法语，如果上述判定为真，那么下述哪项判定必定为真。        
```

答案：如果小陈懂英语，那么他一定不懂法语。     

解题思路：

非 (懂英语 and 懂法语)

题干逻辑结构：非 (A and B) = 非 A or 非 B = A -> 非 B = B -> 非 A   

### 考点三 原命题与逆否命题等价

把一个逻辑箭头的两边命题分别加非，并且把逻辑箭头方向取反得到的命题，就是原命题的逆否命题。   

1) A -> B 的逆否命题是 非 A <- 非 B       
2) B -> A 的逆否命题是 非 B <- 非 A        
3) B <- A 的逆否命题是 非 B -> 非 A    

```
除非调查，否则没有发言权。以下各项都符合题干的断定，除了        
```

答案：如果调查，就一定有发言权。      

解题思路：    

非调查 -> 非发言权 逆否命题成立：发言权 -> 调查       

题干逻辑结构：非 A -> 非 B = B -> A     

```
有人对 "不到长城非好汉" 这句名言的理解是："如果不到长城，就不是好汉"，假定这种理解为真，则下列哪项判断必然为真。           
```

答案：如果是好汉，他一定到过长城。    

解题思路：

非长城 -> 非好汉 逆否命题成立：好汉 -> 长城   

题干逻辑结构：非 A -> 非 B = B -> A    

### 考点四 逻辑只能顺推不能逆推

A -> B 成立，加上 B -> C 成立，可以得到 A -> C 成立。    

逻辑箭头可以顺推不能逆推。    

[大圈套小圈，逻辑真与事实真相结合判断]
[肯前肯后，否后否前]  

A -> B 成立 + A 为真 = 可以得到 B 为真    
A -> B 成立 + B 为真 = 什么也推不出来       
A -> B 成立 + B 为假 = 可以得到 A 为假       
非 B -> 非 A 成立 + 非 B 为真 = 非 A 为真    

```
如果王林是公司经理，那么他一定学过管理学；王林没有学过管理学，如果上述命题是真的，必然能推出的是：        
```

答案：王林不是公司经理。     

解题思路：

公司经理 -> 管理学 + 非管理学 = 非公司经理       

题干逻辑结构：A -> B 成立 + B 为假 = 可以得到 A 为假    

### 逻辑思维误区

#### 逻辑真与事实真的区别

事实真：给出的条件能够唯一确定事实情况。    

A 真。  小明参加会议。    
非 A 为真。 小明没有参加会议。     
A and B 为真。  小明和小李都参加了会议。     

逻辑真：给出了两者关系，但不能唯一确定事实情况。    

A -> B 为真。 如果小明参加了会议，那么小李也会参加会议。    
A or B 为真。  小明参加会议或者小李参加会议。     

#### A -> B 和 B -> A 的区别

A = B 属于等价关系，即 A -> B 成立，B -> A 也成立。     

注：逻辑箭头是单向的，并不等价，即 A -> B 和 B -> A 并不严格等价。        

#### 为什么看起来是逻辑真，却得出事实真的意思

山无棱、天地合、才敢与君绝。   

山无棱 and 天地合 -> 与君绝     
非与君绝 -> 山有棱 or 天地不合    

#### A -> B 为真，为什么有时候看起来，非 A 却能推出非 B      

如果我带了钥匙，那么我就能进家门     

A -> B 成立，并且 A 是 B 的唯一原因时，非 A 为真，非 B 才为真。    

> 在逻辑中，任何事情都是互相交织，多因多果的。没有单独给出条件的情况下，不能脑补 A 是 B 的唯一原因。  

### 总结

1) A -> B，非 B -> A，非 A or B 是完全相同的三个逻辑关系     

如果 A，那么 B。只有 B，才 A，都等于 A -> B。         

2) A -> B 的逻辑为真，并不等于 A、B 为真 (逻辑真 != 事实真)      

A 和 非 A 相互矛盾，但 A -> B 和 A -> 非 B 不矛盾。     

3) A -> B 为真，+ A 为真，可以推出 B 为真。+ 非 A 为真，什么也推不出。
