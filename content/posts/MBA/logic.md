### 考官出题套路范例

#### 前真后假思路真题范例    

```
小张说：[如果]我中了500万，我[就]送你套房子，哪个选项为真，能证明小张说的话是[错误]的。   
```
答案：中了500万，没送你房子    

解题思路：     

中了500万 -> 送你套房子     

题干逻辑结构：A -> B         
问使其错误的选项：A and 非B          

```
[只有]具有一定文学造诣[且]具有生物学专业背景的人，[才]能读懂这篇文章。如果上述命题为真，哪个选项不可能为真。    
```

答案：从未学过生物学知识的小李读懂了这篇文章     

解题思路：

文学造诣 and 生物学背景 <- 读懂这篇文章

题干逻辑结构：B and C <- A             
问不可能为真的选项：A and (非B or 非C)              

```
[如果]不注重成本的控制，[那么]就无法提供有吸引力的发射价格。
[如果]想在商业发射市场占据一席之地，[那么]就必须提供有吸引力的发射价格。
若上述论述为真，以下哪个选择一定为假？
```

答案：一个小型火箭即占据发射市场的一席之地，同时又不注重成本的控制      

解题思路：    

1) 一席之地 -> 有吸引力的发射价格   A -> B    
2) 非注重成本 -> 非有吸引力的发射价格 等价于 有吸引力的发射价格 -> 注重成本    
3) 非C -> 非B | B -> C （逆否）    
4) A -> B -> C     
5) 一席之地 -> 有吸引力的发射价格 -> 注重成本    
6) 问不可能为真的选项：A and 非C     

```
[所有]值得拥有专利的产品或设计方案[都是]创新。
[所有]的模仿都[不是]创新。
若上述论述为真，以下哪个选择一定为假？
```

答案：有些值得拥有专利的产品是模仿的。     

解题思路：

1) 拥有专利的产品 -> 创新   A -> B     
2) 模仿 -> 非创新 等价于 创新 -> 非模仿           
3) C -> 非B | B -> 非C （逆否）     
4) A -> B -> C        
5) 拥有专利的产品 -> 创新 -> 非模仿       
6) 问不可能为真的选项：A and 非C

## 形式逻辑基础知识

### 形式逻辑基础

#### 运算符

* 非 (NOT)：非A，非B     
* 或 (OR)：A or B，A or 非B，B or A    
* 且 (AND)：A and B，A and 非B，非B and A    
* 推出 (->)：A -> B，A -> 非B，B <- A (B 被 A 推出)      

1) 选言命题：A 和 非A 是矛盾关系，一定一真一假。若 A 为真，那么 非A 为假，反之亦然。给出一个命题为真，等同于给出它的否命题为假。       
2) 选言命题：A or B 的文字描述：A 或者 B，A 和 B [至少有一个]。A 和 B 至多有一个去 = 至少有一个不去 = 非A or 非B。       
3) 联言命题：A and B 的文字描述："也" -- 我是...， xxx也是；"逗号" -- 我是，xxx是；"并且" -- 收入增高并且幸福感增加；"但是" -- 他胜利了，但他不开心。     

### 选言联言命题真假判断

```
若中国应该大力发展农业为真，
试着判断以下4个命题是真？是假？或者不能确定真假？

1) 中国应该大力发展农业或者大力发展工业    A or B 真    
2) 中国应该大力发展农业并且大力发展工业    A and B 不能确定    
3) 中国不应该大力发展农业，也不应该大力发展工业  非A and 非B 假    
3) 或者中国不应该大力发展农业，或者不应该大力发展工业  非A or 非B 不能确定    
```

### 假言命题真假判断     

#### 定义

##### 充分条件假言命题

即有保证。在逻辑学上：如果 A 的发生一定能保证 B 的发生，那么 A 就叫做 B 的充分条件，逻辑表达式为 A -> B。   

例句：如果摩擦一定会生热    
例句：如果断电电灯泡一定会熄灭    

在语言中一般用下面的逻辑连词来描述：   

[如果] A。。。[那么] B。。。     
[只要] A。。。[就] B。。。    
[若] A。。。[则] B。。。    
[所有的] A，[都是] B。。。  

##### 必要条件假言命题

即必须要有的。    

在逻辑学上，如果要得出 B 条件，A 条件必须成立，但 A 条件成立，也不足以证明 B。     

那么，A 就是 B 的必要条件，表达为 A <- B。    

ps: 箭头指向 [必须] / [基础]   

例句：[要想] 下雨 [必须] 天上有云。  天上有云 <- 下雨 (* 箭头指向必须 *)    
例句：[只有] 有足够氧气，人 [才] 能生存。    足够氧气 <- 人才能生存    
例句：一个国家 [要想] 保持政治稳定，它所创造的财富 [必须] 得到公正的分配。 公正分配 <- 政治稳定
例句：维护好经济秩序，是经济快速发展的 [基础]。   经济秩序 <- 快速发展   

在语言中一般用下面的逻辑连词来描述：        

[只有] A。。。[才] B。。。(*)       
A 对于 B 是 [必须] 的 / [不可缺少] 的    
A 是 B 的 [基础] / [必要条件] / [重要前提]     

##### 充要条件假言命题

在逻辑学上，当 A 同时即是 B 的充分条件，又是 B 的必要条件的时候，A 叫做 B 的充分必要条件，简称为充要条件。     

在语言中一般用下面的逻辑连词来描述：    

A 发生 [当且仅当] B 发生。    

等同于：A -> B 和 B -> A 同时成立。如果 A，那么 B。只有 A，才 B。        

### 假言命题逻辑本质

A -> B 什么情况下为假？   [仅在] A and 非B(前真 and 后假)    前 -- 箭头发起方向    
A <- B 什么情况下为假？   [仅在] 非A and B(前真 and 后假)    前 -- 箭头发起方向     

充分条件描述 A -> B 真值表：    

| A | B   | A -> B   |
|--|-----|-----|
| 真 | 真 | 真 |
| 真 | 假 | 假 |
| 假 | 真 | 真 |
| 假 | 假 | 真 |

```
小张说：如果我中了500万，我就送你套房子。      
小李说：你说的是假话，在骗人。       
哪个选项最能表述小李的意思：    
```

答案：我中了500万，没送你房子。      

必要条件描述 A <- B 真值表：

| A | B   | A <- B   |
|--|-----|-----|
| 真 | 真 | 真 |
| 真 | 假 | 真 |
| 假 | 真 | 假 |
| 假 | 假 | 真 |

```
小张说：这次我们录取的原则是，只有评分优秀，才能够录取。         
小李说：那么针对小王，你没有遵循这个原则。       
哪个选项最能表述小李的意思：    
```

答案：小王评分不优秀，结果被录取了。    

总结：    

不用区分充分还是必要，它只是一个概念，只是主语的角度不同，要区分逻辑的前件与后件。     

前件 -> 后件    
后件 <- 前件     

一个逻辑，仅在同时满足前件为真 (逻辑箭头发起的方向) 和后件为假 (逻辑箭头结束的方向) 的时候，逻辑才为假。       

### 常见逻辑连词

#### 假言命题常见逻辑连词

当需要判断一个逻辑的真假时，关键是判断逻辑箭头的方向，并不需要判断充分 / 必要条件。    

需要做的事：根据逻辑连词，把文字替换成逻辑表达式。    

常见的逻辑连词与其相对应的逻辑箭头方向。       

1) 充分条件表述形式：   

如果 A，那么 B。   描述为： A -> B      
如果 A，则 B。   描述为： A -> B     
所有的 A，都是 B。   描述为： A -> B           
只要 B，就 A。   描述为： B -> A     

2) 必要条件表述形式：   

只有 B，才 A。   描述为：B <- A      
只有 A，才 B。   描述为：A <- B   

3) 特殊逻辑连词：[除] 非 A，否则 B。 

描述为：非 A -> B    

4) *** 特殊逻辑连词：(否则 B)，除非 A。     

[例子] (否则)肖群一周工作五天，除非这周内有法定节假日。 没有法定节假日 -> 工作五天   

除非不 A，否则 B。 描述为：A -> B    
除非不 A，否则不 B。 描述为：A -> 非 B    
除非 A，否则不 B。 描述为：非 A -> 非 B
B，除非 A。 描述为：非 A -> B    
除非 A，B。 描述为：非 A -> B   

> 优先级：才 > 如果(若) > 除非

### 形式逻辑解题必备能力

只看逻辑连词，不看文字。    

## 形式逻辑四大考点

### 考点一 对逻辑关系加 "非"

1) 非 (非A) = A     
2) 非 (A or B) = 非 A and 非 B    
3) 非 (A and B) = 非 A or 非 B    
4) *** 非 (A -> B) = A and 非 B [前真后假]   

```
小红想买一件漂亮的衣服，她父亲同意或者她母亲同意都会给她买，可是小红没能说服家人给她买这件衣服，说明：    
```

答案：她父母都不同意给她买    

解题思路：    

非 (父亲同意 or 母亲同意)    

题干逻辑结构：非 (A or B) = 非 A and 非 B          

```
一个项目要实施前必须得到总经理和董事长的签字，这个项目没有得到实施，说明：     
```

答案：或者总经理没签字，或者董事长没签字       

解题思路：    

非 (总经理签字 and 董事长签字)      

题干逻辑结构：非 (A and B) = 非 A or 非 B             

### 考点二 or 和 -> 的相互转化   

对于 or 的逻辑，不管左右是什么，对 or 的左边加非，可以推出右边。对 or 的右边加非，可以推出左边。    

1) A or B = 非 A -> B = 非 B -> A (你掏钱 or 我掏钱 = 非你掏钱 -> 我掏钱 = 非我掏钱 -> 你掏钱)    
2) 非 A or 非 B = A -> 非 B = B -> 非 A (小张和小李至多入选一个 = 小张不如选 or 小李不如选)       
3) 非 A or B = A -> B = 非 B -> 非 A   
4) A or 非 B = 非 A -> 非 B = B -> A     
5) 非 (非 A or B) = 非 (A -> B) = A and 非 B   
6) A and B or C = 非 A or 非 B -> C = 非 C -> A and B    

```
小陈并非既懂英语又懂法语，如果上述判定为真，那么下述哪项判定必定为真。        
```

答案：如果小陈懂英语，那么他一定不懂法语。     

解题思路：

非 (懂英语 and 懂法语)

题干逻辑结构：非 (A and B) = 非 A or 非 B = A -> 非 B = B -> 非 A   

### 考点三 原命题与逆否命题等价

把一个逻辑箭头的两边命题分别加非，并且把逻辑箭头方向取反得到的命题，就是原命题的逆否命题。   

1) A -> B 的逆否命题是 非 A <- 非 B       
2) B -> A 的逆否命题是 非 B <- 非 A        
3) B <- A 的逆否命题是 非 B -> 非 A    

```
除非调查，否则没有发言权。以下各项都符合题干的断定，除了        
```

答案：如果调查，就一定有发言权。      

解题思路：    

非调查 -> 非发言权 逆否命题成立：发言权 -> 调查       

题干逻辑结构：非 A -> 非 B = B -> A     

```
有人对 "不到长城非好汉" 这句名言的理解是："如果不到长城，就不是好汉"，假定这种理解为真，则下列哪项判断必然为真。           
```

答案：如果是好汉，他一定到过长城。    

解题思路：

非长城 -> 非好汉 逆否命题成立：好汉 -> 长城   

题干逻辑结构：非 A -> 非 B = B -> A    

### 考点四 逻辑只能顺推不能逆推

A -> B 成立，加上 B -> C 成立，可以得到 A -> C 成立。    

逻辑箭头可以顺推不能逆推。    

[大圈套小圈，逻辑真与事实真相结合判断]
[肯前肯后，否后否前]  

A -> B 成立 + A 为真 = 可以得到 B 为真    
A -> B 成立 + B 为真 = 什么也推不出来       
A -> B 成立 + B 为假 = 可以得到 A 为假       
非 B -> 非 A 成立 + 非 B 为真 = 非 A 为真    

```
如果王林是公司经理，那么他一定学过管理学；王林没有学过管理学，如果上述命题是真的，必然能推出的是：        
```

答案：王林不是公司经理。     

解题思路：

公司经理 -> 管理学 + 非管理学 = 非公司经理       

题干逻辑结构：A -> B 成立 + B 为假 = 可以得到 A 为假    

### 逻辑思维误区

#### 逻辑真与事实真的区别

事实真：给出的条件能够唯一确定事实情况。    

A 真。  小明参加会议。    
非 A 为真。 小明没有参加会议。     
A and B 为真。  小明和小李都参加了会议。     

逻辑真：给出了两者关系，但不能唯一确定事实情况。    

A -> B 为真。 如果小明参加了会议，那么小李也会参加会议。    
A or B 为真。  小明参加会议或者小李参加会议。     

#### A -> B 和 B -> A 的区别

A = B 属于等价关系，即 A -> B 成立，B -> A 也成立。     

注：逻辑箭头是单向的，并不等价，即 A -> B 和 B -> A 并不严格等价。        

#### 为什么看起来是逻辑真，却得出事实真的意思

山无棱、天地合、才敢与君绝。   

山无棱 and 天地合 -> 与君绝     
非与君绝 -> 山有棱 or 天地不合    

#### A -> B 为真，为什么有时候看起来，非 A 却能推出非 B      

如果我带了钥匙，那么我就能进家门     

A -> B 成立，并且 A 是 B 的唯一原因时，非 A 为真，非 B 才为真。    

> 在逻辑中，任何事情都是互相交织，多因多果的。没有单独给出条件的情况下，不能脑补 A 是 B 的唯一原因。  

### 总结

1) A -> B，非 B -> A，非 A or B 是完全相同的三个逻辑关系     

如果 A，那么 B。只有 B，才 A，都等于 A -> B。         

2) A -> B 的逻辑为真，并不等于 A、B 为真 (逻辑真 != 事实真)      

A 和 非 A 相互矛盾，但 A -> B 和 A -> 非 B 不矛盾。     

3) A -> B 为真，+ A 为真，可以推出 B 为真。+ 非 A 为真，什么也推不出。      

## 秒杀思路：前真 and 后假

前真后假秒杀题目题干特性：    

1) 题干问法为：以下哪个选项一定为假，以下哪个选项能推翻上述论证。       
2) 题干通过逻辑连词，给出 A -> B。        

前真后假秒杀题目正确答案：A and 非 B    

> 注意：此处是 and!!!

```
什么情况下 A -> B 为假 
```

答案：A and 非 B [前真后假]      

```
什么情况下 A <- B and C 为假 
```

答案：B and C and 非 A [前真后假]   

```
如果小张和小李都去参加会议，那么小王和小赵至多有一个人去参加会议     
```

小张 and 小李 -> 非小王 or 非小赵 [至多有一个去 = 至少有一个不去]     

答案：小张 and 小李 and 小王 and 小赵    

```
只有天气晴朗并且温度适宜，旅游者们才会去出海或者潜水     
```

天气晴朗 and 温度适宜 <- 出海 or 潜水    

答案：出海 or 潜水 and (非天气晴朗 or 非温度适宜)

### 总结

1) 同时满足前真后假才为假。   
2) A or B -> C 为真，A and 非 C、B and 非 C 都为假，随便一个条件满足就行。           
3) A and B 为真，A -> 非 B 为假。       
4) A and 非 B 为真，A -> B 为假。     
5) 证明是错的，则找前真后假。      
6) 证明符合，则排除满足前真后假的不符合选项。       

## 秒杀思路：OR 的逻辑

1) A 和 B 中只要有一个为真，A or B 的逻辑就为真。    
2) 仅仅给出 A or B 为真的情况，有三种可能：A 真 B 假、A 假 B 真、A 真 B 真。   
3) 在 A or B 为真的情况下，增加非 A 为真的条件就能得到 B 为真，增加非 B 为真的情况，就能得到 A 为真。    
4) 能够区分 A or B (非 A -> B)（至少有一个），与非 A or 非 B（至少有一个不去 = 至多有一个去）（A -> 非 B）不同。

## 秒杀思路：不相容性选言命题

相容性选言命题两个分支可以同时为真 A OR B    
不相容性选言命题两个分支必须一真一假  A OR^ B   

| A | B   | A OR B   | A OR^ B   |
|--|-----|-----|-----|
| 真 | 真 | 真 | 假 |
| 真 | 假 | 真 | 真 | 
| 假 | 真 | 真 | 真 |
| 假 | 假 | 假 | 假 |

### 不相容性选言命题的逻辑连词

[要么 A，要么 B] : 不相容性选言命题    
[不是 A，就是 B] : 相容性选言命题    

### 相容性选言命题 vs 不相容性选言命题

不相容性选言命题等于多了一个条件。   

[相容性选言命题： A or B 为真]  

1) 非 A -> B 为真，非 B -> A 为真      

[不相容性选言命题：A or B 为真]   

1) 非 A -> B 为真，非 B -> A 为真   
2) A -> 非 B 为真，B -> 非 A 为真   

> 相容性选言命题的推理规则（一边为假推出另一边为真）。不相容性选言命题也适用。
> *** 同时不相容性选言命题还增加了另一个推理的条件（一边为真，可以推出另一边为假）。

```

## 忽略文字看逻辑本质

1) 不管题干用什么逻辑连词来描述一个逻辑，都先把它转换成逻辑箭头的形式。    
2) 对于 or 的逻辑，否定一边可以推出另一边。
A or B 等价于 非 A -> B 等价于 非 B -> A，这三个表达式没有任何区别。   
3) 对于一个逻辑箭头，A -> B 等价于 非 B -> 非 A，这两个表达式没有任何区别。
4) A -> B 和 A and 非 B 互为矛盾命题。即 A -> B 为真，就意味着 A and 非 B 为假。A -> B 为假，就意味着 A and 非 B 为真。     

```
对于 "除非 A，否则 B"，这个描述如何表述为逻辑箭头：  

小张说：应该表述为：非 A -> B         
小李说：应该表述为：非 B -> A         
小王说：应该表述为：A or B           
上述三个人到底谁说的是正确答案呢？                     

A、小张说得对      
B、小李说得对     
C、小王说得对      
D、三个人都不对       
E、三个人说的都对                    
```

答案：E   

```
为什么所有的 A 都是 B，等同于 A -> B?   
```

小圈套大圈，A 是小圈，B 是大圈。   

*** [有的 A 是 B] A and B    
*** [有的 A 不是 B] A and 非 B    
*** [有的 B 不是 A] B and 非 A  

总结：    

以下四种情况都是 A -> B    

1) A -> B:   
如果 A，那么 B   
如果 A，则 B   
只要 A，就 B  
除非不 A，否则 B   
所有的 A 都是 B    

2) B <- A    
B 是 A 的基础   
B 是 A 的必要条件   
只有 B，才 A   
必须有 B，才有 A   

3) 非 B -> 非 A   
如果不 B，那么不 A   
只要不 B，就不 A   
没有 B，就没有 A   
除非 B，否则不 A     

4) 非 A or B    
或者不 A，或者 B   
并非 (A and 非 B)   
非 （有的 A 不是 B）    

## 形式逻辑常见思维误区

### 混淆逻辑真和事实真

```
小李说：如果我们努力学习，就能够考上满意的大学。   
小王说：如果我们努力学习，就不能够考上满意的大学。

A、小王的话为真

D、小王和小李考上了满意的大学                      
E、小王的话真假不能确定                     
```

小李：A -> B 不等价于 A 真 B 真   
小王：A -> 非 B 不等价于 A 真 B 假   

答案：E

1) 如果明天去游泳，那么明天就不去唱歌

第二句话的逻辑为：非 B -> A，转换为 or 的逻辑为：A or B

## 复合命题考官出题套路 

### 考官套路一：代入逻辑推命题真假

1) 题干通过逻辑连词给出数个逻辑箭头，一定还会给出某个命题的真假的条件。   
2) 选项均是要判断命题的真假。      
 

> 否定 or 的一边，可以推出另一边   

逻辑真 != 事实真 != A 真 B 真    

A -> B 真 + A 真 = B 真  

> 逻辑只能顺推，前件为真，可以推出后件为真。前件为假，推不出任何信息。   

A -> B 成立 + A 为真，可以推出 B 为真。   
A <- B 成立 + B 为真，可以推出 A 为真。   
A -> B 成立 + A 为假，A <- B 成立 + B 为假，推不出任何信息。    

> 逻辑不能逆推，后件为真，推不出任何信息。后件为假，可以推出前件为假(逆否命题成立)。   

A -> B 成立 + B 为真，A <- B 成立 + A 为真，推不出任何信息。     
A <- B 成立 + A 为假，可以推出 非 B 为真。   
A -> B 成立 + B 为假，可以推出 非 A 为真。  

> A or B 为真，否定 or 的一边可以推出另一边，肯定一边什么也推不出。    

A or B 成立 + 非 A 为真，可以推出 B 为真。    
A or B 成立 + 非 B 为真，可以推出 A 为真。    
A or B 成立 + A 为真，A or B 成立 + B 为真，推不出任何信息。

某花店只有从花农那里购得低于正常价格的花才能以低于市场的价格卖花而获利。    
除非是该花店的销售量很大，否则，不能从花农那里购得低于正常价格的花。   
要想有大的销售量，该花店就要满足消费者的兴趣或者拥有特定品种的独家销售权。   
如果上述断定为真，则以下哪项必定为真？   

A、如果该花店从花农那里购得低于正常价格的花，那么就会以低于市场的价格卖花而获利   
B、如果该花店没有以低于市场的价格卖花而获利，则一定没有从花农那里购得低于正常价格的花   
C、该花店不仅满足了消费者的个人兴趣，而且拥有特定品种独家销售权，但仍然不能以低于市场的价格卖花而获利   
D、如果该花店广泛满足了消费者的个人兴趣或者拥有特定品种独家销售权，那么就会有大的销售量    
E、如果该花店以低于市场价格卖花而获利，那么一定是从花农那里购得了低于正常价格的花                                                            
```

低于正常价格进货 <- 卖花获利   
非(销售量很大) -> 不能低于正常价格购花   
满足兴趣 or 独家销量 <- 大销售量  

卖花获利 -> 低于正常价格进货 -> 大量销售 -> 满足兴趣 or 独家销量   

答案：E
