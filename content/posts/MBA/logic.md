## 导学

### 逻辑考试大纲分析 

#### 学习逻辑的意义

* 更好的理解世界

#### 结构化思维

逻辑考察的是结构化思维，只认准逻辑连词(或者、只有、并且、所以、才...等)，不要像阅读理解似的做题。     

A or B    
A and B       

A and B <- C    
A or B -> C

### 备考建议和备考误区

#### 备考关键

* 体会考官思路，切换思维模式。    
* 总结和归纳方法，而不是用感觉或日常文字理解做题。    

#### 解题方法

* 形式逻辑本质上仅有4个基本考点
* 论证推理本质上仅有7种最基本套路
* 结论推出本质上只需要5种排除方法
* 分析推理本质上仅有2种出题套路

#### 逻辑复习步骤及方法

* 看考点精讲的书 / 视频，理解套路
* 真题 (2-3 遍)
* 再看一遍书 / 视频
* 模考题

#### 备考误区

* 用日常思维去阅读理解而不是用逻辑思维去分析题目结构
* 过度追求 “技巧” 而忽略了对题目逻辑结构的分析
* 不总结，闷头刷题，追求做题数量的成就感
* 盲目迷信模拟题、押题

#### 三要三不要

* 不要盲目做题，先看一遍思路，有了思路的概念再做题体会思路
* 不要做模拟题，优先做真题 (从最近的开始做)
* 不要孤立的做题，要思考不同题干文字下隐藏的逻辑结构 (逻辑连词)

### 考官出题套路范例

#### 前真后假思路真题范例    

```
小张说：[如果]我中了500万，我[就]送你套房子，哪个选项为真，能证明小张说的话是[错误]的。   
```
答案：中了500万，没送你房子    

解题思路：     

中了500万 -> 送你套房子     

题干逻辑结构：A -> B         
问使其错误的选项：A and 非B          

```
[只有]具有一定文学造诣[且]具有生物学专业背景的人，[才]能读懂这篇文章。如果上述命题为真，哪个选项不可能为真。    
```

答案：从未学过生物学知识的小李读懂了这篇文章     

解题思路：

文学造诣 and 生物学背景 <- 读懂这篇文章

题干逻辑结构：B and C <- A             
问不可能为真的选项：A and (非B or 非C)              

```
[如果]不注重成本的控制，[那么]就无法提供有吸引力的发射价格。
[如果]想在商业发射市场占据一席之地，[那么]就必须提供有吸引力的发射价格。
若上述论述为真，以下哪个选择一定为假？
```

答案：一个小型火箭即占据发射市场的一席之地，同时又不注重成本的控制      

解题思路：    

1) 一席之地 -> 有吸引力的发射价格   A -> B    
2) 非注重成本 -> 非有吸引力的发射价格 等价于 有吸引力的发射价格 -> 注重成本    
3) 非C -> 非B | B -> C （逆否）    
4) A -> B -> C     
5) 一席之地 -> 有吸引力的发射价格 -> 注重成本    
6) 问不可能为真的选项：A and 非C     

```
[所有]值得拥有专利的产品或设计方案[都是]创新。
[所有]的模仿都[不是]创新。
若上述论述为真，以下哪个选择一定为假？
```

答案：有些值得拥有专利的产品是模仿的。     

解题思路：

1) 拥有专利的产品 -> 创新   A -> B     
2) 模仿 -> 非创新 等价于 创新 -> 非模仿           
3) C -> 非B | B -> 非C （逆否）     
4) A -> B -> C        
5) 拥有专利的产品 -> 创新 -> 非模仿       
6) 问不可能为真的选项：A and 非C      

#### 引入前提思路真题范例    

```
林教授患有支气管炎，为了取得疗效，张医生[要求]林教授[必须戒烟]。以下哪项是张医生的[要求所假设]的。
```

答案：林教授抽烟。       

解题思路：

## 形式逻辑基础知识

### 形式逻辑基础

#### 运算符

* 非 (NOT)：非A，非B     
* 或 (OR)：A or B，A or 非B，B or A    
* 且 (AND)：A and B，A and 非B，非B and A    
* 推出 (->)：A -> B，A -> 非B，B <- A (B 被 A 推出)      

1) 选言命题：A 和 非A 是矛盾关系，一定一真一假。若 A 为真，那么 非A 为假，反之亦然。给出一个命题为真，等同于给出它的否命题为假。       
2) 选言命题：A or B 的文字描述：A 或者 B，A 和 B [至少有一个]。A 和 B 至多有一个去 = 至少有一个不去 = 非A or 非B。       
3) 联言命题：A and B 的文字描述："也" -- 我是...， xxx也是；"逗号" -- 我是，xxx是；"并且" -- 收入增高并且幸福感增加；"但是" -- 他胜利了，但他不开心。     

#### 选言联言命题真假判断

```
若中国应该大力发展农业为真，
试着判断以下4个命题是真？是假？或者不能确定真假？

1) 中国应该大力发展农业或者大力发展工业    A or B 真    
2) 中国应该大力发展农业并且大力发展工业    A and B 不能确定    
3) 中国不应该大力发展农业，也不应该大力发展工业  非A and 非B 假    
3) 或者中国不应该大力发展农业，或者不应该大力发展工业  非A or 非B 不能确定    
```

#### 假言命题真假判断     

##### 定义

###### 充分条件假言命题

即有保证。在逻辑学上：如果 A 的发生一定能保证 B 的发生，那么 A 就叫做 B 的充分条件，逻辑表达式为 A -> B。   

例句：如果摩擦一定会生热    
例句：如果断电电灯泡一定会熄灭    

在语言中一般用下面的逻辑连词来描述：   

如果 A。。。那么 B。。。     
只要 A。。。就 B。。。    
若 A。。。则 B。。。    
所有的 A，都是 B。。。  

###### 必要条件假言命题

即必须要有的。在逻辑雪上，如果要得出 B 条件，A 条件必须成立，但 A 条件成立，也不足以证明 B。那么，A 就是 B 的必要条件，表达为 A <- B。    

例句：下雨必须天上有云。  天上有云 <- 下雨    
例句：只有有足够氧气，人才能生存。    足够氧气 <- 人才能生存     

在语言中一般用下面的逻辑连词来描述：    

只有 A。。。才 B。。。       
A 对于 B 是必须的 / 不可缺少的    
A 是 B 的基础 / 必要条件 / 重要前提     

###### 充要条件假言命题

在逻辑学上，当 A 同时即是 B 的充分条件，又是 B 的必要条件的时候，A 叫做 B 的充分必要条件，简称为充要条件。     

在语言中一般用下面的逻辑连词来描述：    

A 发生[当且仅当] B 发生。    

等同于：A -> B 和 B -> A 同时成立。如果 A，那么 B。只有 A，才 B。        

##### 假言命题逻辑本质

A -> B 什么情况下为假？   A and 非B(前真 and 后假)    前 -- 箭头发起方向    
A <- B 什么情况下为假？   非A and B(前真 and 后假)    前 -- 箭头发起方向     

充分条件描述 A -> B 真值表：    

| A | B   | A -> B   |
|--|-----|-----|
| 真 | 真 | 真 |
| 真 | 假 | 假 |
| 假 | 真 | 真 |
| 假 | 假 | 真 |

```
小张说：如果我中了500万，我就送你套房子。      
小李说：你说的是假话，在骗人。       
哪个选项最能表述小李的意思：    
```

必要条件描述 A <- B 真值表：

| A | B   | A <- B   |
|--|-----|-----|
| 真 | 真 | 真 |
| 真 | 假 | 真 |
| 假 | 真 | 假 |
| 假 | 假 | 真 |

```
小张说：这次我们录取的原则是，只有评分优秀，才能够录取。         
小李说：那么针对小王，你没有遵循这个原则。       
哪个选项最能表述小李的意思：    
```

答案：小王评分不优秀，结果被录取了。    

总结：    

不用区分充分还是必要，要区分逻辑的前件与后件。     

前件 -> 后件    
后件 <- 前件     

一个逻辑，仅在同时满足前件为真 (逻辑箭头发起的方向) 和后件为假 (逻辑箭头结束的方向) 的时候，逻辑才为假。
